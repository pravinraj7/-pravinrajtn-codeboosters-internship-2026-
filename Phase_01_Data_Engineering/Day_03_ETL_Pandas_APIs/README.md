# Weather Data ETL Project

## Objective
Fetch live weather data using OpenWeatherMap API, clean the data using Pandas, and save it as CSV.

## Technologies Used
- Python
- Pandas
- Requests
- Google Colab

## ETL Process

### Extract
Collected weather data from OpenWeatherMap API.

### Transform
Converted JSON data into structured DataFrame.

### Load
Stored cleaned data into CSV file.

## Output
Generated `weather_data.csv`

# Employee Salary ETL Project

## Objective
Load employee salary data from Excel, clean duplicate records, calculate yearly salary, and export cleaned data.

## Technologies Used
- Python
- Pandas
- Google Colab

## ETL Process

### Extract
Read employee data from Excel file.

### Transform
- Removed duplicate records
- Calculated yearly salary

### Load
Exported cleaned dataset into CSV file.

## Output
Generated `cleaned_employee_salary.csv`.
